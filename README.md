## Getting Started

### Prerequisites
* nodejs
* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/melogtm/wikiday
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run server, you can set up another port if ```3000``` is being used
   ```sh
   node index.js
   ```
